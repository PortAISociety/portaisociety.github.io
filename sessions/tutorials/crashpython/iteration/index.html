Summary

A list is an ordered set of data, all of the same type. You create a list by defining a variable as being equal to a set of open and closed brackets. To access the data inside a list, you reference the index of the data you want. The append and remove functions are used to add and remove values from a list respectively.

A for loop is used to iterate over a data set a finite number of times to perform actions on the data within the data set.

Examples


# Declaration examples
hopes_and_dreams = []

shopping = ["apples", "bleach", "cable ties"]

important_numbers = [42, 69, 420, 34]
    

# Referencing examples
print(important_numbers[0])
print(important_numbers[2])
    

# Appending example
hopes_and_dreams.append("to be happy")

# Removing example
shopping.remove("apples")

# Value alteration example
shopping[0] = "oranges"
    

# Checking for value
if "bleach" in shopping:
print("That's a lot of bleach you've got there...")

if "apples" not in shopping:
print("You really need to eat more healthily")
    

# Iteration example
for item in shopping:
print(item)
    

# Complex iteration example
for x in range(0, len(shopping)):
print(x, shopping[x])
