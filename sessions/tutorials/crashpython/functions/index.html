Summary

A function is a named store of code, allowing that code to be used multiple times across a program. You define a function using the ‘def’ keyword, and call it by using its name. Parameters allow you to pass data into a function using variables named in parentheses when you define the function. The return keyword allows you to get data out of a function.

Examples


# Basic function example
def say_hello():
    print("Hello")


# Calling example function
say_hello()
            

# Parameter example
def say_number(number):
    print(number)


number = 5
say_number(42)
print(number)
            

# Return example
def multiply(x, y):
    z = x * y
    return z


i = multiply(5, 3)
print(multiply(3, 2))
            

# Branching return example
def max_of(x, y):
    if x > y:
        return x
    elif x < y:
        return y
    else:
        return 0

Exercises

  1. The force needed to crush a pumpkin is 500N! Write a function that takes the mass and velocity of a falling boulder and works out its force using 0.5 * mass * velocity^2. The function should then return whether or not the boulder would crush the pumpkin.

  2. A group of spoopy skeleton scientists want to know whether the rocks they have could crush a pumpkin depending on their velocity. The boulders range from 10kg to 100kg in 10kg steps, and the skeletons can get them from 0 meters per second up to speeds of 10 meters per second. Create a program that works out which of these boulders could crush a pumpkin and what velocity they would need to be going at.

    Hint: You can nest for-loops inside one another.
