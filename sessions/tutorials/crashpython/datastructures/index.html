Summary

A dictionary is a collection of keys and values that allow you to quickly and directly reference information. Dictionaries are defined using curly brackets, and are referenced in the same way as lists but using keys instead of index numbers.

An object is a store for data and functions. The data is specific to the object and cannot be accessed directly outside of it. An object is defined using the ‘class’ keyword.

Exercises

  1. Santa Claus has to deliver presents to a lot of children on Christmas Night. Each child has a name and age, and is allowed one present, which they have politely written to him to ask for. Santa also needs to keep track of whether they’ve been naughty or nice and whether he’s visited them yet. Create a class to help Santa keep track of all this information.

  2. Once Santa has visited a child, he needs to be able to tick them off his list. Create a function within the class you have just made to mark off when he’s visited that child.

  3. Finally, all these children need to be put on a list for Santa to check once or twice whilst out on deliveries. But a list would be too long to search through. Make a dictionary to hold his list and populate it with children. What could the key be for each child value?
