Part 3: Program Flow

Back to course contents

Summary

A comparator compares two values and returns a boolean value as to whether the statement is true or false. The different comparators are equal to (==), not equal to (!=), less than (<), greater than (>), less than or equal to (<=), and greater than or equal to (>=). If-statements allow you to selectively trigger code and create branching programs. While loops allow you to loop over a section of code until a condition is no longer true. You create an infinite loop by using a condition that is never false for the loop, which is a bad idea.

Examples


# Example if-statement
name = input("What is your name? ")

if name == "Sam":
    print("Hello, Sam!")

elif name == "Carmen":
    print("Welcome, Carmen!")

elif name == "Geralt":
    print("Welcome, Gwynbleidd")

elif name == "Dovahkiin":
    print("I used to be an adventurer like you...")

else:
    print("I don't recognise that name...")

print("Nice to meet you.")
            

# Example while-loop
count = 0
while count <= 10:
    if count == 5:
        print(count)
    else:
        print("hi")

    count = count + 1

print("The loop is broken!")
            

# Example of nested loops
count = 0
i = 0
while count <= 5:
    if count == 5:
        while i < 3:
            print(i)
            i = i + 1
    print("spinning")
    count = count + 1
print("Finished spinning!")
            

# Example of break keyword
count = 0
while True:
    print("Help!")
    count = count + 1
    if count == 20:
        break
print("Thanks for saving me!")
