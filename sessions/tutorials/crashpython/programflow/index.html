Summary

A comparator compares two values and returns a boolean value as to whether the statement is true or false. The different comparators are equal to (==), not equal to (!=), less than (<), greater than (>), less than or equal to (<=), and greater than or equal to (>=). If-statements allow you to selectively trigger code and create branching programs. While loops allow you to loop over a section of code until a condition is no longer true. You create an infinite loop by using a condition that is never false for the loop, which is a bad idea.
