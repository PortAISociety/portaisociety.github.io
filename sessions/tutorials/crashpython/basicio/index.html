Summary

The print function allows you to output information to the shell. This can be hard-coded data, or it can be the contents of a variable. A variable is a temporary, named store for data that is disposed of once the program is finished. You can use the input function to get user input and store it in a variable.

Examples


# The print function
print("hello")

# Defining variables
number = 42
string = "Password123"
not_a_number = "42"

# Getting user input
name = input("What is your name? ")

Exercises

  1. In the file you created, create a program which defines a variable with your name stored in it and prints it. Run the program.

  2. Create a program which asks the user their name and says hello to them.

  3. The ideal number of pockets a person should have is always 2 more than what they already have. Write a program that asks the user how many pockets they currently have and tells them how many they should have.
